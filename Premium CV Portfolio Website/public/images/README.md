# Image Assets

This folder contains image assets for the website.

## Structure

- `profile.jpg` or `profile.png` - Your profile photo (recommended size: 400x400px)
- `tech/` - Tech stack icon images
  - `python.svg` or `python.png`
  - `langchain.svg` or `langchain.png`
  - `aws.svg` or `aws.png`
  - `azure.svg` or `azure.png`
  - `docker.svg` or `docker.png`
  - `kubernetes.svg` or `kubernetes.png`
  - `terraform.svg` or `terraform.png`
  - `powerbi.svg` or `powerbi.png`

## Usage

Images in the `public` folder are accessible at `/images/` in the deployed site.

For example:
- Profile photo: `/images/profile.jpg`
- Python icon: `/images/tech/python.svg`
